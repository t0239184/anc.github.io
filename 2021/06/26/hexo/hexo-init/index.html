    前言

    有段時間沒用Hexo, 忘記如何啟動, 如何Deploy
    而當時的對於前端技術也不太熟悉, 有很多東西都是照著步驟做的
    但現在對於前端熟悉了, 重新看了Hexo的官網, 才漸漸回憶起來
    所以先寫起來, 以後可以參照

    安裝相關依賴

    將blog從git上Clone下來後先將相關依賴裝回來

    npm install

    下載hexo-cli

    npm install -g hex-cli

    Hexo的基本操作

    然後可以用以下幾個指令進行操作

    • hexo clean
    • hexo generate
    • hexo server
    • hexo deploy

    NPM已經封裝好的指令

    但現在在package.json中已經將上面這些操作封裝起來了
    所以可以使用以下指令

    • npm run build
    • npm run server
    • npm run deploy
