    CN  Common Name         : 網域名稱
O   Organization        : 組織名稱
OU  Organizational Unit : 部門 default:Common Name
L   Locality            : 城市名稱
ST  State               : 州或郡
C   Country             : 國家

    DN似乎是有順序的, 而且和LDAP有關
    IBM的說明有提到DN是有順序的, 特別是OU
