Docker Volume

Volume 可以滿足需要互相交流檔案的需求或者留存資料用途，例如資料共享和資料庫檔案等

Docker 提供三種掛載類型(mount types)：

Mount Type Descript
Bind mount Mount exist host filesystem
Volume Mount volume instance
Tmpfs mount Mount from memory

建議增加Volume，Container必須移除重新生成會比較好

1.bind mount

2.volume Instance

3. tmpfs mount
