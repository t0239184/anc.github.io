Homebrew Load Error

這是某次 MacOS Mojave 更新之後結果 HomeBrew 無法使用的問題，看錯誤訊息是 Ruby 那邊的問題，所以測試了一下

Install 無法執行

Update 可以執行

Upgrade 無法執行

Doctor 無法執行

網路上查有很多解法，其中一個解法就是重裝，但多數都是移除HomeBrew，然後再安裝一次，但這個做法會使得原本安裝的所有套件及應用程式等等被刪除，所以我認為這不是一個好的做法，於是嘗試了一種作法，重裝。

結論嘗試執行安裝程式，看來可以執行，直接覆蓋原有的檔案，解決。
